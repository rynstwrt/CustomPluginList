COMMANDS:
- /custompluginlist: Show a list of this plugin's commands.
- /cpl help: Same as /custompluginlist.
- /cpl seemessage: See the message players see when attempting to run /plugins without the bypass node.
- /cpl seemsg: Same as /cpl seemessage.
- /cpl reload: Reload the plugin's config file.

PERMISSIONS:
- cpl.help: Permission to run /cpl or /cpl help.
- cpl.seemessage: Permission to use /cpl seemsg.
- cpl.reload: Permission to run /cpl reload.
- cpl.bypass: Users with this node will see the real plugin list when running /plugins.
